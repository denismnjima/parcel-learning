# Parcel Learning

This project is a simple example of how to use Parcel to bundle a JavaScript project.

## Getting Started

To get started, you need to have Node.js installed on your machine. You can download it from [nodejs.org](https://nodejs.org/).

Once you have Node.js installed, you can install Parcel by running the following command:

```bash
npm install -g parcel-bundler
```

## Running the Project  

To run the project, you can use the following command:

```bash
npm run dev
```

This will start the development server and open your default browser to view the project.

## Building the Project

To build the project, you can use the following command:

```bash
npm run build
```

This will create a `dist` folder with the bundled JavaScript file.
